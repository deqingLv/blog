Mac 提效工具推荐

# 窗口管理工具

## Rectangle

Rectangle 是一个免费开源的窗口管理工具，可以快速调整窗口大小和位置。

### 安装

```
brew install --cask rectangle
```

### 常用快捷键

- `⌘ + ⌥ + ←` : 左半屏
- `⌘ + ⌥ + →` : 右半屏
- `⌘ + ⌥ + ↑` : 上半屏
- `⌘ + ⌥ + ↓` : 下半屏
- `⌘ + ⌥ + F` : 全屏
- `⌘ + ⌥ + C` : 居中

## Magnet

Magnet 是另一个优秀的窗口管理工具，功能类似 Rectangle，但需要付费。

```
brew install --cask magnet
```

# 启动器和搜索工具

## Raycast

Raycast 是一个现代化的启动器，可以替代 Spotlight，功能强大且免费。

### 安装

```
brew install --cask raycast
```

### 主要功能

- 快速启动应用
- 文件搜索
- 剪贴板历史
- 扩展插件系统
- 计算器、单位转换等快捷工具

## Alfred

Alfred 是老牌的 macOS 启动器，功能非常强大，但 Powerpack 需要付费。

### 安装

```
brew install --cask alfred
```

### 主要功能

- 应用启动和文件搜索
- 工作流（Workflows）系统
- 剪贴板历史
- 系统命令快捷操作

# 剪贴板工具

## Paste

Paste 是一个美观的剪贴板历史管理工具。

```
brew install --cask paste
```

### 功能特点

- 保存所有剪贴板历史
- 快速搜索历史记录
- 支持图片、文本等多种格式
- 快捷键快速访问
